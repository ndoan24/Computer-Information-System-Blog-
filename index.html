How does the Internet Work?

Technology, October 29, 2020

  1. 1. In your own words, what is a network?
    2.

    A network is a communication between computers that uses the same communication protocol


  2. 2. What’s the difference between the ‘web’ and ‘the internet’?
    3.

    The difference between a “web” and the “internet” is that the web is websites that have a single document or multiple pages and content. The internet is using IP addresses, and world wide connection.


  3. 3. The internet is not a physical place or chip, so how would you explain it?
    4.

    The internet is made up of two computers are more that are connected to a server, IP address that communicates with a server to open up a web that has a domain that has content


  4. 4. How is ISP related to the internet?
    5.  How ISP (internet service provider) is related to the internet is by providing the bandwidth and speed of internet access. For example, comcast xfinity. There are faster providers and slower ones. It uses a network that communicates to each other to have access to web pages, and other content.
  5. 5. Why was the internet created, when?
    6.

    The internet was created to have a better form of communication and reaching a broader audience. The purpose was for communication purposes, also it originally started to help out military bases. It started out with emails, and people used emails like crazy when it first came out.


  6. 6. What is DNS and why is it necessary?
    7.

    DNS is the domain name. A domain name is a name of a website. It is needed to identify and go to a certain webpage, or website. It is necessary to be able to look up a certain website for more information. For example, businesses, accounts, and more.


  7. 7. How has the internet expanded in the last 10 years in relation to computers?
    8.

How the internet has expanded in the last 10 years has been how many users use the internet now, and how it is needed for literally everything. 10 years ago, it was used for common tasks such as addresses, emails, games, computer art. Now it has advanced to use to learn new things, communicate to loved ones overseas, meeting new people, and meeting virtually online. Learning how to use the internet is essential for everyone and has been taught in schools, and how to use a computer and keyboard. The Internet is available on all devices now, not just computers.

    Meal Delivery Cart Robot

    Robot, Oct. 27, 2020

    Meal Delivery Cart Robot

    The input for the meal delivery cart is the speaker where it can talk and respond to customers. It can also play music as well when food gets delivered it celebrates with music. The meal delivery cart is also voice activated like Alexa but 2000x better because this is the future. The output is the recording of when the meal delivery carts gets to and from the location of resturant to customer's location. Another output is the GPS location where customers and resturants can keep track of the food. The Central Processing Unit (CPU) is the data and memory work together within the RAM inside the meal delivery cart. The little square processor inside the robot is the "brain". It requires memory to buffer GPS tracking, text to speech, and music playback. The arithmetic and logic unit (ALU) is the calculations the robot does in the GPS to keep track of how long the delivery will take. The control unit is the processor inside the meal delivery cart. This processor tells the robot how to respond to text-to-speech, how fast to move, and to censor when it is safe to cross the street with stop light movement. This is also controlled by the person watching the robot behind the scenes.
